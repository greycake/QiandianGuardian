# 黔电卫士 - 贵州电话拦截器

## 功能说明

- ✅ 自动拦截所有贵州地区的固定电话和手机号码
- ✅ 拦截贵州地区短信
- ✅ 短信完全拦截（不显示在收件箱）
- ✅ 电话自动挂断（使用CallScreeningService）
- ✅ 适配鸿蒙4.3系统
- ✅ 查看拦截记录
- ✅ 拦截开关控制

## 兼容性

- Android 7.0+
- 鸿蒙4.0+
- 已测试：华为Mate X2 (鸿蒙4.3.0.190)

## 权限说明

首次使用需要授权以下权限：
- 电话权限（拦截来电）
- 短信权限（拦截短信）
- 通话记录权限（记录拦截历史）
- 设置为默认通话应用（完全控制来电）

## 拦截效果

- **电话**：来电短暂响铃1-2秒后自动挂断，标记为"已拦截"
- **短信**：完全拦截，不会在收件箱显示

## 构建方法

### 方式1：GitHub Actions（推荐）

1. 将代码推送到GitHub仓库
2. GitHub Actions会自动构建APK
3. 在Actions页面下载生成的APK

### 方式2：本地构建

```bash
./gradlew assembleRelease
```

APK位置：`app/build/outputs/apk/release/app-release.apk`

## 安装说明

1. 下载APK文件
2. 在鸿蒙系统上安装
3. 打开应用，授予所需权限
4. 开启拦截开关
5. 设置为默认通话应用（系统会提示）

## 注意事项

- 首次使用需要授予多项权限
- 鸿蒙系统可能需要手动开启"自启动权限"
- 短信拦截需要在系统设置中设置为默认短信应用
